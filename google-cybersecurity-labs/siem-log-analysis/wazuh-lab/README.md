# Wazuh SIEM Deployment & Alert Analysis Lab
## Objective
    Deploy Wazuh SIEM to monitor endpoint logs and detect suspicious activity.

## Environment
    Ubuntu 22.04 VM (Wazuh Server)
    Ubuntu 22.04 VM (Monitored Endpoint)

## Setup
    Installed Wazuh Manager
    Installed and registered Wazuh Agent
    Configured log collection

## Activities Performed
    Generated failed SSH login attempts
    Ran Nmap scan on localhost

## Alerts Observed
    Authentication failure alerts
    Suspicious command execution logs

## Skills Demonstrated
    SIEM deployment
    Agent configuration
    Log analysis
    Threat detection
    Security event investigation
